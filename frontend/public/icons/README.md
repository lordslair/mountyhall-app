# PWA Icons

This directory should contain PNG icons for the Progressive Web App:

- `icon-192x192.png` - 192x192 pixels
- `icon-512x512.png` - 512x512 pixels

Currently, an SVG placeholder (`icon.svg`) is provided. To complete the PWA setup, replace the SVG with proper PNG icons.

You can generate icons using:
- Online tools (e.g., https://realfavicongenerator.net/)
- Image editing software
- Command line tools like ImageMagick or PIL

The icons should represent the MountyHall app branding.
